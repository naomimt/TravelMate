# TravelMate

TravelMate is a simple, responsive, and user-friendly static website designed to showcase travel destinations and provide information about the travel services offered. This project is built using **HTML** and **CSS**.

## Features

- **Home Page**: Overview of the TravelMate service with featured destinations.
- **Destinations Page**: Display of popular travel destinations with images and descriptions.
- **About Page**: Information about TravelMate and its mission.
- **Contact Page**: A form to contact TravelMate (static, no backend functionality yet).
- **Responsive Layout**: Designed to work on desktops, tablets, and mobile devices.
